Interface LocaleValues

The locale values used to localize phrases.

Hierarchy

  • LocaleValues

Indexable

[phrase: string]: string | object

Localized phrases indexed by their corresponding localizable phrase. These phrases are formatted using sprintf.js. Phrases with a quantifier have additional mappings by multiplicity of the quantifier. See the following for more details on the format: https://github.com/alexei/sprintf.js#format-specification.
